# MFCV - موقع سيرة ذاتية تفاعلي

## الملفات
- `index.html` صفحة السيرة الذاتية العامة.
- `admin.html` لوحة التحكم.
- `style.css` التصميم.
- `app.js` عرض البيانات.
- `admin.js` إدارة البيانات.

## التشغيل
افتح `index.html` مباشرة أو ارفع الملفات إلى GitHub Pages / Vercel.

## الدخول
افتح `admin.html`.
كلمة المرور الافتراضية:
`123456`

## ملاحظة مهمة
هذه النسخة تحفظ البيانات والملفات داخل `localStorage` في المتصفح. هذا يعني أن التعديلات تظهر على نفس المتصفح والجهاز فقط، ولا تُزامن تلقائيًا مع كل زوار الموقع.

لتحويله إلى موقع حي حقيقي يمكن تعديله من أي جهاز، نحتاج في المرحلة التالية إلى ربطه بقاعدة بيانات وتخزين ملفات مثل Supabase أو Firebase، مع نظام تسجيل دخول حقيقي.

## GitHub Pages
ارفع الملفات الخمسة إلى المستودع، وتأكد أن `index.html` في المجلد الرئيسي، ثم فعّل GitHub Pages.

## Vercel
اربط مستودع GitHub بالمشروع في Vercel، وسيتم نشر `index.html` تلقائيًا.
